## Quick Start

### Installation
```bash
pip install -r requirements.txt
```

### Running the Dashboard

**Default port (8501):**
```bash
streamlit run streamlit_dashboard.py
```

**Custom port:**
```bash
streamlit run streamlit_dashboard.py --server.port 8080
```

**Using the script with custom port:**
```bash
./run_dashboard.sh 8080
```

The dashboard will automatically open in your browser at `http://localhost:<PORT>`

## Tips for Best Experience

1. **Run the Notebook First**: For full functionality, run the Jupyter notebook to generate:
   - Trained model (for accurate predictions)
   - SHAP values (for explainability)
   - Optimization results CSV

2. **Data Requirements**: Ensure `Synthetic_Farming_Dataset_With_Seasonality_And_Challenge.csv` is in the same directory

3. **Optimization Results**: The optimization page requires `optimized_input_recommendations.csv` (generated by the notebook)

4. **Interactive Features**: 
   - Use filters to narrow down results
   - Hover over charts for detailed information
   - Download data for further analysis

5. **Performance**: The dashboard uses caching for faster loading. Clear cache if data is updated.

## Troubleshooting

### Dashboard won't start
- Check if all dependencies are installed: `pip install -r requirements.txt`
- Ensure Python 3.8+ is being used
- **Port conflicts**: If port 8501 is in use, specify a different port:
  ```bash
  streamlit run streamlit_dashboard.py --server.port 8080
  ```
  Or use the script: `./run_dashboard.sh 8080`

### Missing data errors
- Ensure the CSV file is in the correct location
- Check file names match exactly (case-sensitive)

### Model predictions not working
- Run the Jupyter notebook first to train the model
- The dashboard uses placeholder predictions if model is not available

### Optimization results not showing
- Run the optimization section in the notebook
- Check if `optimized_input_recommendations.csv` exists
